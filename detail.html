Winzy - Chat App

About the Project

Winzy is a real-time chat app designed for fast and secure communication. Developed using SwiftUI and Firebase Realtime Database, the app allows users to chat with each other instantly, providing a smooth user experience and sleek design.

Key Features
  • Real-time chat
  • User authentication with Firebase
  • Group and private chats
  • Message notifications
  • Profile management
Technology Stack
  • SwiftUI
  • Firebase Realtime Database
  • Swift
  • Cloud Storage
  • Push Notifications
